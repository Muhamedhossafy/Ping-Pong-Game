# Ping-Pong-Game
Ping Pong Game

**Ping Pong Game** is a classic two-player game where players compete to score points by hitting a ball past their opponent. The game features:

**Scoring**: The player on the right scores one point each time the ball passes the player on the left and the player on the left side is similar in the rules.

**Game Over**: The game ends when either player reaches a score of seven.
